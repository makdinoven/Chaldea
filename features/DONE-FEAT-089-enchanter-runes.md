## 1. Feature Brief (filled by PM — in Russian)

### Описание
Зачарователь может создавать руны по рецептам (уже работает через базовую систему крафта) и вставлять/извлекать их в оружие и броню. Система слотов уже реализована в FEAT-086 (ювелир), но работает только для украшений + камней. Нужно расширить: зачарователь работает с оружием/бронёй + рунами.

### Что уже есть (из FEAT-086)
- `socket_count` на предметах (задаётся в админке)
- `socketed_gems` JSON на character_inventory/equipment_slots (per-instance)
- Эндпоинты: socket-info, insert-gem, extract-gem
- `build_modifiers_dict` учитывает бонусы из слотов
- Equip/unequip переносит данные слотов

### Что нужно изменить
Текущие ограничения в коде:
- `insert-gem` проверяет `item_type in JEWELRY_TYPES` (ring, necklace, bracelet) — нужно добавить оружие/броню для зачарователя
- `insert-gem` проверяет `gem_item.item_type == 'gem'` — нужно добавить `'rune'` (новый item_type)
- `extract-gem` аналогично ограничен украшениями
- Профессия проверяется как `jeweler` — нужно также разрешить `enchanter`

### Механика
- **Зачарователь** создаёт руны по рецептам из магических материалов (эссенций от алхимика и т.д.)
- Руны — предметы item_type = "rune", с модификаторами статов (как камни)
- Зачарователь вставляет руны в оружие и броню (head, body, cloak, belt, main_weapon, additional_weapons, shield)
- Зачарователь извлекает руны с шансом сохранения по рангу (как ювелир с камнями)
- Ювелир по-прежнему работает только с украшениями + камнями

### Разделение по профессиям
| Профессия | Предметы для слотов | Вставляет | item_type вставки |
|-----------|-------------------|-----------|------------------|
| Ювелир | ring, necklace, bracelet | Камни | gem |
| Зачарователь | head, body, cloak, belt, main_weapon, additional_weapons, shield | Руны | rune |

### Извлечение рун (шанс по рангу)
Те же шансы что у ювелира:
- Ученик (ранг 1): 10% сохранение
- Подмастерье (ранг 2): 40% сохранение
- Мастер (ранг 3): 70% сохранение

### Бизнес-правила
- Только зачарователь может вставлять/извлекать руны
- Руны только в оружие/броню, камни только в украшения
- Система слотов универсальна — `socketed_gems` хранит и камни, и руны (по item_id)
- socket-info должен показывать руны для зачарователя, камни для ювелира
- Руны крафтятся по рецептам (уже работает)

### UX
- Зачарователь открывает таб "Крафт"
- Видит секцию "Руны и слоты" (аналогично "Камни и слоты" у ювелира)
- Выбирает оружие/броню с свободными слотами
- Вставляет руну, видит бонусы

### Edge Cases
- Что если ювелир пытается вставить руну? → Ошибка "Только зачарователь может вставлять руны"
- Что если зачарователь пытается вставить камень? → Ошибка "Зачарователь может вставлять только руны"
- Что если предмет не подходит для профессии? → Ошибка "Руны можно вставлять только в оружие и броню"
